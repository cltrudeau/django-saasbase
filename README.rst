django-saasbase
***************

Django app that helps with basic user management functions for a SaaS web-site


Installation
============

.. code-block:: bash

    $ pip install django-saasbase

Supports
========

django-saasbase has been tested with:

* Django 2.1.2 using Python 3.6, 3.7
